## Description

White Rabbit coding test

## Installation

Clone this repo, go to project folder then run

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Production

To run the project in prodution mode, run

```bash
# build
$ npm run build

# run
$ npm run start:prod
```

## After that please visit

http://localhost:3000
